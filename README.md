# Portfolio Website using Flask

## Objective
Create a personal portfolio website using Flask.

## Tools Used
- Python
- Flask
- HTML
- CSS

## Features
- Personal information section
- Skills section
- Contact form
- Flask routing

## How to Run

1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000

## Outcome
A simple portfolio website built using Flask.
